Senior backend Developer for MMOG

Description

A game studio, which is working on an innovative title in a close partnership with the world-leading publisher, is looking for a backend developer. As a Server Game Developer, you’ll be responsible for implementing server-related game features. You’ll be working closely with the server team to create scalable infrastructure as well as the client team for feature integration. You’ll have to break out of your toolset to push boundaries on technology to deliver the most robust backend to our users.

Main tasks and responsibilities

  • Development of APIs for the client-server interaction
  • Design and development of a server-side game backend
  • Close collaboration with the client development team
  • Development and support of backend game data editors
  • Development and support of internal analytical services
  • Social networking integration

Requirements

  • Experience with MMOGs or comparable high-loaded client-server technologies
  • Strong knowledge of the C# programming language
  • Strong knowledge of the PostgreSQL and object/relational DBMS technologies
  • Passion for programming
  • Good communication and interpersonal skills - must be a team-player
  • Good English skills
  • Desire to work on ambitious projects in a small flexible team
  • Desire to learn, improve skills
  • High motivation and focus on result

Great Pluses

  • Familiarity with social networks services and technologies
  • Knowledge of PHP and other basic web technologies (HTML, CSS)
  • Design patterns knowledge
  • Passion for video games
  • Version control software familiarity (Git, SVN)
  • Bug-tracking software familiarity (Jira, Redmine)
  • Motivation to teach others
